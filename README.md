# Gemini Chat App

A modern AI chat application built with React and Gemini API.

## Technologies
- React
- JavaScript
- CSS
- Gemini API

## Features
- AI-powered chat interface
- Chat history with local storage
- Responsive design
- Dark/light mode support
- Voice input support
- Typing animation effect

## Environment Variables

Create a `.env` file in the root folder and add your own API key:

REACT_APP_GEMINI_API_KEY=your_api_key_here

## How to Run

npm install  
npm start  

## Status

This project is under development.
